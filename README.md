## Introduction

This application is a react ecommerce application built with React js

### Technologies used

- React JS
- React Router
- Firebase
- Redux
- Payment using Stripe Api
- Css in Js (Styled Components)
- Redux Saga - A scalable way for us to handle asynchronous actions
- React Hooks
- Context Api
- PWA
